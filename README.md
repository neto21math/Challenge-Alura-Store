#  Análisis Exploratorio de Ventas - Alura Store

##  Propósito del Proyecto

Realizar un **Análisis Exploratorio de Datos (EDA)** para comparar el rendimiento, las tendencias de venta y las métricas clave de cuatro sucursales (Tiendas 1,2,3,4). El objetivo es identificar la tienda menos rentable.

## Estructura de Archivos

- `Alura_Store_Análisis.ipynb`: Notebook de Jupyter con el código Python completo (análisis y visualizaciones).
- `README.md`: Este documento.

##  Insights Clave (Ejemplos)



> **Gráfico:** Se incluye un gráfico de barras comparativo que muestra la marcada diferencia de rendimiento entre las Tiendas.

## Instrucciones de Uso

1.  **Clonar el repositorio.**
2.  Abrir el archivo `Alura_Store_Análisis.ipynb` en **Google Colab** o Jupyter Notebook.
3.  Asegurarse de que el archivo de datos esté cargado en la ruta correcta (`data/`).
4.  Ejecutar todas las celdas del Notebook en orden.
